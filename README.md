# Develop

外部Asssetsを配置する

以下からUnityChanをダウンロード（ver 1.3.0)
https://unity-chan.com/download/releaseNote.php?id=UnityChan

以下からOVRLipSyncをダウンロード（v29)
https://developer.oculus.com/downloads/package/oculus-lipsync-unity/

`/Assets`に配置する。

```
- Assets/
    - Oculus/
        ...
    - UnityChan/
        ...
```
